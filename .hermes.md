# Grimoire

A character sheet creation application for the homebrew TTRPG **Divergence**.

## Before Working

- Read `IDEA.md` for the project vision (short).
- Read `DESIGN.md` for the full design spec (detailed — ruleset, features, data model, scope). This is the source of truth for what to build.
- `DESIGN.md` may not exist yet. If it doesn't, ask the user before proceeding.

## Conventions

- This is a brand-new project — no existing code yet. Start clean.
- Commit frequently with clear messages (`feat:`, `fix:`, `docs:`, `chore:`).
